Service Description

The service is a web application that lets you submit a web site or web page for quality tests.

The service performs selected tests against the content of one or more web pages, collects errors, warnings and recommendations and presents you with relevant reports.

Test Data Retrieval

To enable normal operation in the testing of a web site, the service may attempt to retrieve the web site's:

  • robots.txt

  • sitemap.xml

  • sitemap.txt

  • sitemap as defined in robots.txt and/or any discovered sitemaps

  • sitemaps as defined within discovered sitemaps

  • RSS or Atom feed as defined in the <head> section of the web site's root

  • web page content

  • linked resources as defined in the markup of a web page

Test Data Storage

When a single web page test is carried out, the service stores the fully-resolved URL of the page tested and the results of the selected tests for that URL.

When a full web site test is carried out, the service stores a list of URLs tested and the results of the selected tests for each URL.

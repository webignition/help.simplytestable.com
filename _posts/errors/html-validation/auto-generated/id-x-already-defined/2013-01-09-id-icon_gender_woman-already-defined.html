Most Likely Cause and Fix

The value icon_gender_woman has already been used for an id attribute.

Find all attribute occurrences of the form … id="icon_gender_woman" … and ensure each has a unique value within the given document.

What This Error Means

The id attribute is used to uniquely identify a single element within a document: 

<img src="http://example.com/header.png" id="header-image" >

In order to uniquely identify a single element, the value of the id attribute must be unique within the given document.

This error is telling you that the value icon_gender_woman has already been used for an id attribute.

How To Fix It

Find all attribute occurrences of the form … id="icon_gender_woman" … and ensure each has a unique value within the given document.

For example, use: 

<div id="icon_gender_woman-foo">
  <p>Foo</p>
</div>

<div id="icon_gender_woman-bar">
  <p>Bar</p>
</div>

instead of: 

<div id="icon_gender_woman">
  <p>Foo</p>
</div>

<div id="icon_gender_woman">
  <p>Bar</p>
</div>

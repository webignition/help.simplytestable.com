Most Likely Cause and Fix

You've used an invalid value for the name attribute on a meta element.

The WHAT WG wiki lists available registered meta extensions for HTML5 documents.

HTML4 documents are not restricted on the values that can be used. Check that you're using the correct document type declaration.

What This Error Means

Some element attributes can take pretty much any value, others can take only values of a certain type and others still can only take a value from a pre-defined set.

This error relates to the following usage in your markup: 

<meta name="twitter:card" />

The error is stating that, in this context, the attribute name on element meta is not allowed to take the value twitter:card

The validator is giving the reason: "keyword twitter:card is not registered."

How To Fix It

For name attributes on meta elements in HTML5 documents

The WHAT WG wiki lists available registered meta extensions for HTML5 documents.

Check that you're using a registered value.

For name attributes on meta elements in HTML4 documents

HTML4 documents are not restricted on the values that can be used. Check that you're using the correct document type declaration.

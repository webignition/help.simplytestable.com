Most Likely Cause and Fix

Your markup contains more than one blank ID attribute in the form: 

<div id="">

Ensure all ID attributes have a non-empty value: 

<div id="foo">

What This Error Means

The id attribute is used to uniquely identify a single element within a document: 

<img src="http://example.com/header.png" id="header-image" >

In order to uniquely identify a single element, the value of the id attribute must be unique within the given document.

This error is telling you that the value <blank> has already been used for an id attribute.

How To Fix It

Find all attribute occurrences of the form … id="" … and ensure each has a unique value within the given document.

For example, use: 

<div id="foo">
  <p>Foo</p>
</div>

<div id="bar">
  <p>Bar</p>
</div>

instead of: 

<div id="">
  <p>Foo</p>
</div>

<div id="">
  <p>Bar</p>
</div>

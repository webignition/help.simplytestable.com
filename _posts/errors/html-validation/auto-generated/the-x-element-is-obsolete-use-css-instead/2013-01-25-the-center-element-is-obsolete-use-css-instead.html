Most Likely Cause and Fix

Your markup makes use of a <center> element which is no longer allowed in HTML.

Use the CSS text-align property to achieve the same effect.

Use: 

<div style="text-align:center">
  This text will be centered.
  <p>So will this paragraph.</p>
</div>

instead of: 

<center>
  This text will be centered.
  <p>So will this paragraph.</p>
</center>

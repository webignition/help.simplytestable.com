Most Likely Cause and Fix

Your markup makes use of a <big> element which is no longer allowed in HTML.

Use the CSS font-size property to achieve the same effect.

Use: 

<p>
  This is the first sentence. <span style="font-size:1.2em">This whole 
  sentence is in bigger letters.</span>
</p>

instead of: 

<p>
  This is the first sentence. <big>This whole 
  sentence is in bigger letters.</big>
</p>

What This Error Means

The <big> element is no longer allowed in HTML.

This error is alerting you to this fact and informing you that an equivalent end result can be achieved using CSS.

How To Fix It

Use the CSS font-size property to achieve the same effect.

Use: 

<p>
  This is the first sentence. <span style="font-size:1.2em">This whole 
  sentence is in bigger letters.</span>
</p>

instead of: 

<p>
  This is the first sentence. <big>This whole

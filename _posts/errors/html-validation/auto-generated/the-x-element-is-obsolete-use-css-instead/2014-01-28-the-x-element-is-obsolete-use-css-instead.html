Most Likely Cause and Fix

Your markup makes use of a <X> element which is no longer allowed in HTML.

Remove this element and achieve an equivalent effect with CSS.

What This Error Means

The <X> element is no longer allowed in HTML.

This error is alerting you to this fact and informing you that an equivalent end result can be achieved using CSS.

Most Likely Cause and Fix

Your markup makes use of a <strike> element which is no longer allowed in HTML.

Use the CSS text-decoration property to achieve the same effect.

Use: 

<span style="text-decoration:line-through;">Hello World!</span>

instead of: 

<strike>Hello World!</strike>

What This Error Means

The <strike> element is no longer allowed in HTML.

This error is alerting you to this fact and informing you that an equivalent end result can be achieved using CSS.

Use the CSS text-decoration property to achieve the same effect.

Use: 

<span style="text-decoration:line-through;">Hello World!</span>

instead of: 

<strike>Hello World!</strike>

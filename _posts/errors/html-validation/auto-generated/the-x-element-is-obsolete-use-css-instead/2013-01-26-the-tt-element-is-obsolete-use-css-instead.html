Most Likely Cause and Fix

Your markup makes use of a <tt> element which is no longer allowed in HTML.

Remove this element and achieve an equivalent text styling with CSS.

Use: 

<p>Enter the following command: <span style="font-family: monospace;">ping simplytestable.com</span></p>

instead of: 

<p>Enter the following command: <tt>ping simplytestable.com</tt></p>

What This Error Means

The id attribute is used to uniquely identify a single element within a document: 

<img src="http://example.com/header.png" id="header-image" >

In order to uniquely identify a single element, the value of the id attribute must be unique within the given document.

This error is telling you that the value p8990286886520184251 has already been used for an id attribute.

How To Fix It

Find all attribute occurrences of the form … id="p8990286886520184251" … and ensure each has a unique value within the given document.

For example, use: 

<div id="p8990286886520184251-foo">
  <p>Foo</p>
</div>

<div id="p8990286886520184251-bar">
  <p>Bar</p>
</div>

instead of: 

<div id="p8990286886520184251">
  <p>Foo</p>
</div>

<div id="p8990286886520184251">
  <p>Bar</p>
</div>

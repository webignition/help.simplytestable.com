What This Error Means

There is a defined set of attributes for each HTML element. Some are required for your markup to be considered valid.

You have omitted the required name attribute on an element that requires it.

How To Fix It

The specific error message you encountered will refer to a line and column number. This will guide you to the exact point in your markup to which the error relates.

For param elements

You've most likely forgotten to add an name attribute to a <param> element: 

<param …>

Add a name attribute to each <param> element where missing: 

<param name="example" …>

In general

Find the relevant element and add the missing name attribute.

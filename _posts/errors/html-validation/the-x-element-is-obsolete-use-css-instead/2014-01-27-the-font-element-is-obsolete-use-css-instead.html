Most Likely Cause and Fix

Your markup makes use of a <font> element which is no longer allowed in HTML.

Remove this element and achieve an equivalent text styling with CSS.

Use: 

<p style="font-size:3em; color:red;"></p>

instead of: 

<p><font size="3" color="red">Hello World!</font></p>

What This Error Means

The <font> element is no longer allowed in HTML.

This error is alerting you to this fact and informing you that an equivalent end result can be achieved using CSS.

How To Fix It

Remove this element and achieve an equivalent text styling with CSS.

Use: 

<p style="font-size:3em; color:red;"></p>

instead of: 

<p><font size="3" color="red">Hello World!</font></p>

ID "popup5" already defined

Most Likely Cause and Fix

The value popup5 has already been used for an id attribute.

Find all attribute occurrences of the form … id="popup5" … and ensure each has a unique value within the given document.

What This Error Means

The id attribute is used to uniquely identify a single element within a document: 

<img src="http://example.com/header.png" id="header-image" >

In order to uniquely identify a single element, the value of the id attribute must be unique within the given document.

This error is telling you that the value popup5 has already been used for an id attribute.

How To Fix It

Find all attribute occurrences of the form … id="popup5" … and ensure each has a unique value within the given document.

For example, use: 

<div id="popup5-foo">
  <p>Foo</p>
</div>

<div id="popup5-bar">
  <p>Bar</p>
</div>

instead of: 

<div id="popup5">
  <p>Foo</p>
</div>

<div id="popup5">
  <p>Bar</p>
</div>

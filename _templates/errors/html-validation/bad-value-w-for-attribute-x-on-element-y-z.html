How To Fix It

This is not a valid error, it occurred as a result in the HTML validator.

There is no fix that you need to make.

The error relates to an invalid attribute value being used on a specific element.

How this might be fixed often depends on the element, attribute, attribute value and the context in which these are being used.

For form actions

Add a relevant value for the action attribute.

Use 

<form action="http://example.com/" … >

instead of 

<form action="" … >

For name attributes on meta elements in HTML5 documents

The WHAT WG wiki lists available registered meta extensions for HTML5 documents.

Check that you're using a registered value.

For name attributes on meta elements in HTML4 documents

HTML4 documents are not restricted on the values that can be used. Check that you're using the correct document type declaration.

For links

Set a valid value for the attribute {{%1}} used on a link in your markup.

Read up on link relations and the HTML4 link types and HTML5 link type extensions.

In general

Change the relevant attribute value to one that is valid for the context in which it is being used.

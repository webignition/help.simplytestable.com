What This Error Means

The id attribute is used to uniquely identify a single element within a document: 

<img src="http://example.com/header.png" id="header-image" >

In order to uniquely identify a single element, the value of the id attribute must be unique within the given document.

This error is telling you that the value referrer_url has already been used for an id attribute.

How To Fix It

Find all attribute occurrences of the form … id="referrer_url" … and ensure each has a unique value within the given document.

For example, use: 

<div id="referrer_url-foo">
  <p>Foo</p>
</div>

<div id="referrer_url-bar">
  <p>Bar</p>
</div>

instead of: 

<div id="referrer_url">
  <p>Foo</p>
</div>

<div id="referrer_url">
  <p>Bar</p>
</div>
